# Alert

| Prop | Type | Default |
| --- | --- | --- |
| `title` | `string` | – |
| `message` | `string` | required |
| `type` | `"info" \| "success" \| "warning" \| "error"` | `"info"` |
| `actions` | `ReactNode` | – |
| `dismissible` | `boolean` | `false` |
| `onDismiss` | `() => void` | – |
| `variant` | `"banner" \| "inline" \| "floating"` | `"banner"` |
